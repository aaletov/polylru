\section{Введение}

Цель работы - разработать многопоточный кэш с методом вытеснения LRU, сравнить
собственную реализацию с реализациями в библиотеках с открытым исходным кодом.

Для достижения поставленной цели в работе были поставленны следующие задачи:

\begin{enumerate}
  \item Изучить подходы к реализации LRU кэша
  \item Разработать спецификацию интерфейса разрабатываемого класса
  \item Разработать тесты, описывающие поведение разрабатываемого класса
  \item Реализовать класс кэша
  \item Сравнить собственную реализацию с реализациями в свободно доступных библиотеках
\end{enumerate}
