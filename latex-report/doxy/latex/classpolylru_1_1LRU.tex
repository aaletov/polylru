\doxysubsection{Подробное описание}
\subsubsection*{template$<$typename Key, typename Value$>$\newline
class polylru\+::\+LRU$<$ Key, Value $>$}

Данный шаблон класса реализует кэш с методом вытесенения \mbox{\hyperlink{classpolylru_1_1LRU}{LRU}}(Least recently used). 

Элементы хранятся ассоциативно, в формате ключ-\/значение, ёмкость кэша задаётся при создании объекта. Размером кэша считается количество элементов, сохранённое в контейнере. Если размер кэша равен его ёмкости, он считается {\bfseries{заполненным}}. Возраст элементов определяется количеством обращений к кэшу, в течение которого они не были использованы. Добавление элемента с новым ключом в заполненный кэш влечёт за собой вытесенение из кэша по методу \mbox{\hyperlink{classpolylru_1_1LRU}{LRU}}, самый старый элемент удаляется.

Функция \mbox{\hyperlink{classpolylru_1_1LRU_adf74956e314cf45970cdcd6e7e6ed369}{resize(size\+\_\+t)}} позволяет динамически изменить размер контейнера.
